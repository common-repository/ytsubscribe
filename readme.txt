Automatically Add Youtube Subscribe Button Below each Video WordPress Plugin

==Description==
ytSubscribe is a Youtube Subscribe Button jQuery Plugin that automatically add/inserts Youtube Subcribe Button Below each Video in your WordPress Post.

How to Find Youtube Channel Id (https://www.youtube.com/watch?v=kxSWWSJkaMY)

Get More Subscribers on YouTube using ytSubscribe(https://www.youtube.com/watch?v=5nYhGxtaWv8)
